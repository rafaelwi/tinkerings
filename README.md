# Tinkerings
### A repo of small things that I've made or am playing around with

## Contents
- [PyPlaysPokemon](https://github.com/rafaelwi/tinkerings/tree/masterPyPlaysPokemon): A simple Python script that 
  can play Pokemon Red or Blue in an emulator. Just open up an emulator and start the script
- [wikicloud](https://github.com/rafaelwi/wikicloud): A Python program that generates word clouds using Wikipedia 
  articles
- [xkcdownloader](https://github.com/rafaelwi/xkcdownloader): A Python script for bulk downloading xkcd comics
